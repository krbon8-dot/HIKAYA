
تطبيق "حكاية" (Hikaya) هو منصة إبداعية متكاملة مصممة خصيصاً لمؤلفي الروايات، كتاب السيناريو، وصناع الـ "مانهوا" والقصص المصورة. يهدف التطبيق إلى تحويل عملية الكتابة من مجرد سرد نصوص إلى تجربة بصرية وتنظيمية احترافية مدعومة بأحدث تقنيات الذكاء الاصطناعي.
إليك نبذة عن ميزاته وفؤائده:
أبرز ميزات تطبيق "حكاية":
محرر القصة بنظام الكتل (Blocks): يسمح لك ببناء صفحتك بمرونة عالية عبر إضافة نصوص، حوارات ممانهوا بفقاعات كلامية متنوعة، صور، جداول تنظيمية، وخطوط فاصلة.
مساعد الكاتب الذكي (AI Chat): شات مخصص يفهم سياق قصتك وعالمك، يمكنك استشارته في تطوير الحبكة، اقتراح أسماء للشخصيات، أو حل ثغرات السيناريو.
أدوات بناء العوالم (World Building): قسم خاص لتوثيق شخصيات القصة، الأماكن، الخرائط، وحتى قاموس خاص لمفردات عالمك الخيالي، مع ميزة "الإكمال التلقائي" أثناء الكتابة.
تعديل الصور بالذكاء الاصطناعي (Nano Banana): ميزة ثورية تتيح لك تعديل الصور المرفوعة (مثل تغيير الملامح أو النمط الفني) عبر أوامر نصية بسيطة مباشرة داخل المحرر.
تنسيق وطباعة احترافية: دعم كامل للغة العربية والخطوط الأدبية، مع إمكانية تصدير العمل كاملاً إلى ملف PDF جاهز للنشر أو الطباعة بتنسيق يحاكي الروايات الحقيقية.
أدوات الذكاء الاصطناعي المدمجة: تدقيق لغوي فوري، اقتراح تكملة للأحداث، وتوليد محتوى ذكي للجداول والبيانات.
وضع التركيز (Focus Mode): واجهة خالية من المشتتات تساعدك على الغوص في عالمك التخيلي والكتابة لساعات دون انقطاع.
فوائد التطبيق للكاتب:
تسريع الإنتاجية: بفضل أدوات الذكاء الاصطناعي، لن تقف طويلاً أمام "قفلة الكاتب"، فالمساعد دائماً جاهز بالأفكار.
التنظيم الفائق: لن تحتاج لدفاتر خارجية؛ كل تفاصيل عالمك (من العملات المستخدمة إلى صلات القرابة بين الشخصيات) محفوظة في مكان واحد ومرتبطة بنصوصك.
تصور بصري فوري: رؤية الحوارات داخل فقاعات والعمل على توزيع الصور يجعلك تشعر بجمالية العمل قبل نشره، مما يحفز الاستمرارية.
احترافية المخرج النهائي: يضمن لك التطبيق مخرجاً نهائياً (PDF) منسقاً بشكل جذاب لا يحتاج لتدخل من مصممي جرافيك.
باختصار، "حكاية" هو رفيق الكاتب العصري الذي يجمع بين "فن الكتابة" و "قوة التقنية" في مساحة إبداعية واحدة.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `API_KEY` in [.env.local](.env.local) to your API key
3. Run the app:
   `npm run dev`
